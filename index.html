On GitHub Pages, your secure admin layer is your GitHub account. Only people with repository access can update the book spreadsheet.

How to update MythShelf

Edit your book list in Excel or Google Sheets, export it as books.csv, then replace the existing books.csv file in your GitHub repository.

1. Open your spreadsheet and update your books, ratings, tiers and tags.
2. Export or download the sheet as CSV.
3. Rename it exactly books.csv.
4. Upload it to the repository root and commit the change.
5. GitHub Pages will show the updated shelf after the site refreshes.
